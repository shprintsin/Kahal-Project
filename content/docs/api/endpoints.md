## תוכן

רשימת כל נקודות הקצה הזמינות.

### GET /api/maps

החזרת רשימת כל המפות.

```json
{
  "maps": [
    {
      "id": "1",
      "title": "מפת קהילות",
      "slug": "communities-map"
    }
  ]
}
```

### GET /api/layers

החזרת רשימת כל השכבות.

### GET /api/collections

החזרת רשימת כל האוספים.

> **הערה:** נקודות קצה אלו מחזירות תוכן ציבורי בלבד. לגישה לתוכן טיוטה, נדרש אימות.
